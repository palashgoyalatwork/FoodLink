# 🍲 FoodLink

> **Connecting Surplus Food with NGOs to Reduce Hunger and Food Waste**

FoodLink is a Streamlit-powered web application that helps restaurants, caterers, and food donors donate surplus food to NGOs before it goes to waste. NGOs can browse available donations, request the required quantity of meals, and track food availability in real time.

---

## 🌍 Problem Statement

Every day, large quantities of edible food are wasted while millions of people struggle with hunger. FoodLink aims to bridge this gap by creating a simple digital platform that connects food donors with NGOs, ensuring that surplus food reaches those who need it most.

---

## ✨ Features

- 🍽️ Restaurant Dashboard to donate surplus food
- 🤝 NGO Dashboard to browse and request meals
- 🔍 Search food donations by food item
- 📦 Partial meal requests supported
- 📉 Automatic update of remaining meal quantity
- 🗑️ Automatic deletion of expired donations
- 📊 Analytics Dashboard
- 🌱 CO₂ emission savings estimation
- 🍴 Total meals saved counter
- 💾 SQLite database for persistent storage

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/palashgoyalatwork/FoodLink.git
```

Move into the project folder:

```bash
cd FoodLink
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
